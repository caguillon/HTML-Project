Let's build our first program!

  1. Open Eclipse
  2. Click File
  3. Click New
  4. Select Java Project
  5. Set project name to be "Hello"
  6. Click Finish
You have just created your first project!! Let's add a file to start coding.
  1. In Eclipse, right click on the project folder ("Hello")
  2. Click New
  3. Click Class
  4. Set the name to be "Main"
  5. Checkmark "public static void main(String[] args)"
  6. Click Finish

Your code should look like this:
package Hello;
public class Main {
public static void main(String[] args) {
// TODO Auto-generated method stub
}
}

Type the following in between the brackets of the main method:
System.out.println("Hello, ARMY!");
Save your code and click on the Run button to see your code in action.

Oracle has a great breakdown of our first project if you want to take a look at additional information!
