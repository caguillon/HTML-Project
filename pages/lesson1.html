Let's build our first program!

  1. Open Eclipse
  2. Click File
  3. Click New
  4. Select Java Project
  5. Set project name to be "Hello"
  6. Click Finish
You have just created your first project!! Let's add a file to start coding.
  1. In Eclipse, right click on the project folder ("Hello")
  2. Click New
  3. Click Class
  4. Set the name to be "Main"
  5. Checkmark "public static void main(String[] args)"
  6. Click Finish
